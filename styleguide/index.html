Introduction

Welcome to Cashew, a living, component-based styleguide of the moving parts that make up the core of GatherContent.

What is this?

Simply put, this styleguide is a repository of components that make up our web application. The components, along with the code that makes up these components, allow us to:

  • Re-use code that has been through endless QA and is tried and tested.
  • Carefully analyse and eliminate design and code consistencies, by keeping these all in one place.
  • Free up our time to focus on the user-flows and logic that comes with building features, rather than aesthetic preferences.

As our app continues to grow, maintainability will become a nightmare. We needed a way to deal with this in a sane manner before it grew out of our control.

Cashew is less of a brand styleguide, and more of a way for our designers and developers to build and collaborate on things quickly and efficiently.
