    Alerts

    Alerts differ from notifications in that they're often displayed when a piece of information needs to be actioned, whereas notifications are used to display a message once an action has been completed, such as applying a template or deleting an item.

    Default alerts

    Examples

    Hey slacker! It looks like you have content that's past it's due date. Get on it! 

                <div class="alert alert--red">
  <p class="alert__message">
    <i class="fa fa-exclamation-circle"></i>
    <strong>Hey slacker!</strong> It looks like you have content that's past it's due date. Get on it!
  </p>
</div>

    Whoops! We did something wrong, please try again, we promise we'll get it right this time. 

                <div class="alert alert--yellow">
  <p class="alert__message">
    <i class="fa fa-question-circle"></i>
    <strong>Whoops!</strong> We did something wrong, please try again, we promise we'll get it right this time.
  </p>
</div>

    Phew, we're okay! It's all good. Move along now. Nothing to see here, young 'un. 

                <div class="alert alert--green">
  <p class="alert__message">
    <i class="fa fa-check-circle"></i>
    <strong>Phew, we're okay!</strong> It's all good. Move along now. Nothing to see here, young 'un.
  </p>
</div>

    Ping! This alert is blue, which we're pretty sure means you have nothing to worry about. Maybe. 

                <div class="alert alert--blue">
  <p class="alert__message">
    <i class="fa fa-info-circle"></i>
    <strong>Ping!</strong> This alert is blue, which we're pretty sure means you have nothing to worry about. Maybe.
  </p>
</div>

    Alerts with actions

    These alerts are simple to set up, and are to be used to action error messages, such as viewing overdue items. Simply add the .alert--with-btn class to your .alert element, and add a .btn. Easy!

    Examples

    Hey slacker! It looks like you have content that's past it's due date. Get on it!

    Complete this thing 
                <div class="alert alert--red alert--with-btn">
  <p class="alert__message">
    <i class="fa fa-exclamation-circle"></i>
    <strong>Hey slacker!</strong> It looks like you have content that's past it's due date. Get on it!
  </p>
  <a href="#" class="btn btn-default--dark">Complete this thing</a>
</div>

    Whoops! We did something wrong, please try again, we promise we'll get it right this time.

    Retry this thing 
                <div class="alert alert--yellow alert--with-btn">
  <p class="alert__message">
    <i class="fa fa-question-circle"></i>
    <strong>Whoops!</strong> We did something wrong, please try again, we promise we'll get it right this time.
  </p>
  <a href="#" class="btn btn-default--dark">Retry this thing</a>
</div>

    Phew, we're okay! It's all good. Move along now. Nothing to see here, young 'un.

    Stop bothering me, creep!

          </div>
