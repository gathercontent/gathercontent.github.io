    Buttons

    Default buttons

    The default buttons are for actions that are neither positive or negative, they simply complete an action. For instance, showing more recent activity on the Project Overview page.

    Examples

    Light button Dark button 
                <a href="#" class="btn btn-default--light">Light button</a>
<a href="#" class="btn btn-default--dark">Dark button</a>

    Primary buttons

    Primary buttons are used to draw attention to primary actions, such as creating an item or applying a template. Primary buttons must use a colour from the primary palette.

    Examples

    Create Primary Destroy Primary Neutral Primary 
                <a href="#" class="btn btn--create">Create Primary</a>
<a href="#" class="btn btn--destroy">Destroy Primary</a>
<a href="#" class="btn btn--positive">Neutral Primary</a>

    Modifiers

    Modifiers are used to alter shape and size, but never colour.

    Small button modifiers

    Examples

    Create Primary small Destroy Primary small Neutral Primary small 
                <a href="#" class="btn btn--create btn--small">Create Primary small</a>
<a href="#" class="btn btn--destroy btn--small">Destroy Primary small</a>
<a href="#" class="btn btn--positive btn--small">Neutral Primary small</a>
