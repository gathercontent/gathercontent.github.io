    Forms

    100% of information submitted to our platform is done via forms, thus they're super important to keep consistent and recognisable across a plethora of contexts.

    Inputs

    Text input

                <input type="text" class="form__element form__input form__input--inline" placeholder="Item name">

    Text input with label

                <label for="item-name" class="form__label">Item name</label>
<input type="text" class="form__element form__input form__input--inline" placeholder="Item name">

    Text input with helper

    e.g. Article name, web page title 
                <input type="text" class="form__element form__input form__input--inline" placeholder="Item name">
<small class="form__microcopy">e.g. Article name, web page title</small>
