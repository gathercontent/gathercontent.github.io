    • Buttons

    Buttons come in two forms: default and primary.

    Default buttons

    The default buttons are for actions that are neither positive or negative, they simply complete an action. For instance, showing more recent activity on the Project Overview page.

    Example

    Light button

    .btn .btn-d--light

    Light button

    .btn .btn-d--dark

    Primary buttons

    Primary buttons are used to draw attention to primary actions, such as creating an item or applying a template. Primary buttons must use a colour from the primary palette.

    Example

    Create button

    .btn .btn--create

    Destroy button

    .btn .btn--destroy

    Neutral button

    .btn .btn--neutral

    Button modifiers

    Modifiers are used to alter shape and size, but never colour.

    Example

    Create button

    .btn .btn--create .btn--s

    Destroy button

    .btn .btn--destroy .btn--s

    Neutral button

    .btn .btn--neutral .btn--s

    Create button

    .btn .btn--create .btn--l

    Destroy button

    .btn .btn--destroy .btn--l

    Neutral button

    .btn .btn--neutral .btn--l

    Create button

    .btn .btn--create .btn--xl

    Destroy button

    .btn .btn--destroy .btn--xl

    Neutral button

    .btn .btn--neutral .btn--xl
